The steps in the terminal are (source: Codecademy):

  • Init git: git init
  • Fetch and merge changes from the remote: git merge origin/master
  • Create a branch to work on the new feature: git branch [name], git checkout [name]
  • Develop the feature on the branch and commit the changes: git add [file], git commit -m [message]
  • Fetch and merge from remote again: git merge origin/master
  • Push the branch up to the remote for review: git push origin [name]
  • Review and approve on Github
